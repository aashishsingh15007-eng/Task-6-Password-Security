# Task 6: Password Strength and Security Evaluation

## Objective
The objective of this task is to understand the core principles of password security, evaluate password strength against common vulnerability testing tools, and research defensive strategies against password-based attacks.

## Tasks Performed
- **Password Evaluation:** Tested various password combinations (weak vs. strong) using `passwordmeter.com`.
- **Attack Research:** Studied the mechanics of Brute Force and Dictionary attacks.
- **Documentation:** Created a comprehensive report detailing password security best practices.

## Key Learnings
- **Password Complexity:** Importance of mixing character types (uppercase, lowercase, numbers, symbols).
- **Password Length:** Length is a primary factor in resisting automated cracking tools.
- **Defensive Strategies:** Using password managers, MFA (Multi-Factor Authentication), and passphrases.
